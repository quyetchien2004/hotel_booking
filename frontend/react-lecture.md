# React 19 Notes

- `main.jsx`: diem vao cua ung dung.
- `App.jsx`: dinh nghia cac route chinh.
- `contexts/`: luu state dung chung toan app.
- `hooks/`: custom hooks tai su dung duoc.
- `services/`: cau hinh API, socket va cac lop giao tiep backend.

## Quy uoc de xuat

- Components chi lo UI.
- Pages ket hop components va logic theo route.
- Service khong render giao dien.
- Context dung cho auth, theme, cart, room state.
